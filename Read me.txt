this is the documentation of dynamic printing in c

what this program does is it inputs a string including spaces and 
prints one letter at a time starting from 'a' till it reaches the required character
of that index
once it reaches it moves on to the next one till all charactes are matched 
in resluting a cool drop down word fromation 

Waring by default i set the cahracter lenth to 20 so if you intend to input 
greater strings lenght change the limit
